# Sentiment-Analysis

Sentiment Pro – Multi-Modal Sentiment Analysis Platform

Sentiment Pro is a web-based multi-modal sentiment analysis system that analyzes text, voice, and facial expressions to detect emotions, sentiment polarity, sarcasm, and emotional intensity.

## 🚀 Features
- User Login System
- Text Sentiment Analysis
- Voice Sentiment Analysis
- Facial Emotion Detection
- AI-generated responses
- Analysis history
- Modern UI

## 🛠️ Technologies
Frontend:
- HTML5, CSS3, JavaScript

Backend:
- Python (Flask)
- NLTK, TextBlob
- Google Gemini AI
